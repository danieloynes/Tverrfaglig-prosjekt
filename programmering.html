Hva har jeg lært

Programmering har vært en av de største fagene i år. Vi har stort sett hatt fokus på å lage nettsider.
Vi har brukt tre forskjellige programmeringsspråk. HTML brukes for nettsidestruktur, og er det vil lærte først.
Det andre vi lærte er CSS som brukes for stil. Innenfor CSS har vi hatt om undertemaer som FlexBox og Media Query.
Til slutt lærte vi Javascript som brukes for interraktivitet.
